Part 3: Bayesian Optimisation

Task 1

First note that: $$\ f(\underline{x})|\underline{f}_{X_n} = \underline{a}_n \sim \mathcal{N}(\tilde{m},\tilde{k})$$ $$\ f(\underline{x}) - a^{*}|\underline{f}_{X_n} = \underline{a}_n \sim \mathcal{N}(\tilde{m} - a^{*},\tilde{k}) $$ $$\ (f(\underline{X}) - a^*|\underline{f}_{X_n}) = \underline{a}_n = Z\sqrt{\tilde{k}(\underline{X},\underline{X}}) + (\tilde{m}(\underline{X}) - a^*)$$ Where, $\ Z \sim \mathcal{N}(0,1) $.

We want to calculate: $$\ \mathbb{E}[\text{max}(f(\underline{x}) - a^*,0)|\underline{f}_{X_n} = \underline{a}_n] $$ From now on, we will suppress the condtioning on $\ \underline{f}_{X_n} = \underline{a}_n$ $$\ = \mathbb{E}[f(\underline{x})-a^*|f(\underline{x})-a^* > 0]$$ $$\ = \mathbb{E}[Z\sqrt{\tilde{k}(\underline{x},\underline{x}}) + (\tilde{m}(\underline{x}) - a^*)|Z> d], \qquad \text{with } d = \frac{a^* - \tilde{m}(\underline{x})}{\sqrt{\tilde{k}(\underline{x},\underline{x}})}$$ $$\ = \sqrt{\tilde{k}(\underline{x},\underline{x}}) \mathbb{E}[Z|Z>d] + (\tilde{m}(\underline{x}) - a^*)\mathbb{E}[1|Z>d])$$ $$\ = \sqrt{\tilde{k}(\underline{x},\underline{x}})\int_{d}^{\infty}Z \phi(Z)\;dZ + (\tilde{m}(\underline{x}) - a^*)\int_{d}^{\infty} \phi(Z)\;dZ$$ Noticing that $\ Z\phi(Z) = -\phi^{'}(Z)$, $$\ = \sqrt{\tilde{k}(\underline{x},\underline{x}}) \phi(d) + (\tilde{m}(\underline{x}) - a^*)(1-\Phi(d))$$ $$\ = \sqrt{\tilde{k}(\underline{x},\underline{x}}) \phi(-d) + (\tilde{m}(\underline{x}) - a^*)\Phi(-d)$$ $$\ = \sqrt{\tilde{k}(\underline{x},\underline{x}}) \phi \left(\frac{\tilde{m}(\underline{x}) - a^*}{\sqrt{\tilde{k}(\underline{x},\underline{x}})} \right) + (\tilde{m}(\underline{x}) - a^*)\Phi \left(\frac{\tilde{m}(\underline{x}) - a^*}{\sqrt{\tilde{k}(\underline{x},\underline{x}})} \right)$$

expected_improvement <- function(x, x_cond, y_cond, mean_func, kernel_func, ...) {
    posterior <- gp_posterior(x, x_cond, y_cond, mean_func, kernel_func, ...)
    m_tilde <- posterior$mean_vec
    a_star <- max(y_cond)
    k_tilde <- posterior$cov_mat
    inside <- (m_tilde - a_star) / (sqrt(diag(k_tilde)))

    EI <- sqrt(diag(k_tilde)) * dnorm(inside) + (m_tilde - a_star) * pnorm(inside)
    return(list("EI" = EI, "post_mean" = m_tilde, "post_k" = k_tilde))
}

Part 4: Bayesian Cubature

Task 1

Note that the integral has the following properties: $$\ \int_{A}\lambda f(\underline{x})\;d{\underline{x}} = \lambda \int_{A} f(\underline{x})\;d{\underline{x}}, \quad \text{and}$$ $$\ \int_{A}f(\underline{x}) + g(\underline{x})\;d{\underline{x}} = \int_{A}f(\underline{x})\;d{\underline{x}} +\int_{A}g(\underline{x})\;d{\underline{x}} $$ Thus, $\ I(\cdot)$ is a linear operator.

Task 2

Since $\ I(\cdot)$ is a continuous and linear , $\ If$ is also a gaussian process that is entirely defined by its mean vector and covariance matrix which we calculate below: $$\ \mathbb{E}\left[If(\underline{x})\right] = \mathbb{E}\left[ \int_{A} f(\underline{x})\;d{\underline{x}}\right]=\int_{A}\mathbb{E}\left[f(\underline{x})\right]\;d\underline{x} =\int_{A}m(\underline{x})\;d\underline{x} $$ $$\ \text{Cov}(If(\underline{x}),If(\underline{y}) ) = \text{Cov}\left(\int_{A} f(\underline{x})\;d{\underline{x}},\int_{A} f(\underline{y})\;d{\underline{y}} \right) = \int_{A} \int_{A}\text{Cov}(f(\underline{x}),f(\underline{y}))\;d\underline{x}d\underline{y} = \int_{A} \int_{A} k(\underline{x},\underline{y})\;d\underline{x}d\underline{y}$$

Task 3

We know the posterior distribution of a gaussian process is also a gaussian process with mean and covariance as below: $$\ \mathbb{E}\left[f(\underline{x})|\underline{f}_X = \underline{a}\right] = m(\underline{x}) + K_X(\underline{x})^T K_{XX}^{-1}(\underline{a} -\underline{m}_X)$$ $$\ \tilde{k}(\underline{x},\underline{y}) = k(\underline{x},\underline{y}) +K_X(\underline{x})^T K_{XX}^{-1}K_X(\underline{y}) $$ Applying $\ I $ to these gives: $$\ \tilde{\mu}= \mathbb{E}\left[If(\underline{x})|\underline{f}_X = \underline{a}\right] = \int_{A} m(\underline{x}) + K_X(\underline{x})^T K_{XX}^{-1}(\underline{a} -\underline{m}_X) \;d\underline{x} $$ $$\ = \int_{A}m(\underline{x})\;d\underline{x} + \int_{A}K_X(\underline{x})^T K_{XX}^{-1}(\underline{a} -\underline{m}_X)\;d\underline{x}$$ $$\ = \int_{A}m(\underline{x})\;d\underline{x} + \int_{A}K_X(\underline{x})^T\;d\underline{x}\;K_{XX}^{-1}(\underline{a} -\underline{m}_X)$$ $$\ = \int_{A}m(\underline{x})\;d\underline{x} + \left[\int_{A}K_X(\underline{x})\;d\underline{x}\right]^T\;K_{XX}^{-1}(\underline{a} -\underline{m}_X)$$ and,

$$\ \tilde{\sigma ^2} = \text{Cov}(f(\underline{x}),f(\underline{y})\;|\underline{f}_X = \underline{a}) = \int_{A}\int_{A} k(\underline{x},\underline{y}) +K_X(\underline{x})^T K_{XX}^{-1}K_X(\underline{y}) \;d\underline{x}\;d\underline{y}$$$$\ = \int_A\int_A k(\underline{x},\underline{y}) \;d\underline{x}\;d\underline{y} + \int_A\int_A K_X(\underline{x})^T K_{XX}^{-1}K_X(\underline{y}) \;d\underline{x}\;d\underline{y}$$$$\ = \int_A\int_A k(\underline{x},\underline{y}) \;d\underline{x}\;d\underline{y} + \int_A K_X(\underline{x})^T\;d\underline{x}\;K_{XX}^{-1}\int_A K_X(\underline{y})\;d\underline{y}$$$$\ = \int_A\int_A k(\underline{x},\underline{y}) \;d\underline{x}\;d\underline{y} + \left[\int_A K_X(\underline{x})\;d\underline{x}\right]^T\;K_{XX}^{-1}\left[\int_A K_X(\underline{y})\;d\underline{y}\right].$$

Task 4

Let $\ f \sim \mathcal{GP}(m,k)$, where $\ m(x) = c$ and $$\ k(x,y) = \lambda^2\left(1+\nu|x-y|\right)\text{exp}(-\nu|x-y|)$$ with $\ \nu = \frac{\sqrt{3}}{\ell}.\\$
$\ k $ can also be written as: $$\ k(x,y) = \left\{ \begin{array}{ll} \lambda^2 (1 - \nu(x-y))\text{exp}(\nu (x-y)) & 0 < x \leq y \\ \lambda^2 (1 + \nu(x-y))\text{exp}(-\nu (x-y)) & y < x < 1 \\ \end{array} \right. $$ If we want to integrate $\ k(x,y)$ over $\ x \in (0,1) $ we can simply do it piece-wise: $$\ \int_0^1 k(x,y) \;dxdy = \int_0^y\lambda^2 (1 - \nu(x-y))\text{exp}(\nu (x-y))\;dx + \int_y^1 \lambda^2(1 + \nu(x-y))\text{exp}(-\nu (x-y))\;dx $$ Calculating the first integral using integration by parts: $$\ \begin{array}{ll} u = 1-\nu(x-y)& u' = -\nu \\ v' = \text{exp}(\nu (x-y)) & v = \nu^{-1} \cdot \text{exp}(\nu (x-y)) \end{array} $$ $$\ \int_0^y\lambda^2 (1 - \nu(x-y))\text{exp}(\nu (x-y))\;dx = \lambda^2\left[\nu^{-1}(1 - \nu(x-y))\text{exp}(\nu (x-y)) + \nu^{-1}\text{exp}(\nu (x-y))\right]_0^y$$ $$\ = \lambda^2\left[2\nu^{-1} - \text{exp}(-\nu y)(y + 2\nu^{-1})\right] $$ Similarly for the second integral: $$\ \begin{array}{ll} u = 1+\nu(x-y)& u' = \nu \\ v' = \text{exp}(-\nu (x-y)) & v = -\nu^{-1} \cdot \text{exp}(-\nu (x-y)) \end{array} $$ $$\ \int_y^1 \lambda^2(1 + \nu(x-y))\text{exp}(-\nu (x-y))\;dx = \lambda^2\left[-\nu^{-1}(1 + \nu(x-y))\text{exp}(-\nu (x-y)) - \nu^{-1}\text{exp}(-\nu (x-y)) \right]_y^1$$ $$\ = \lambda^2\left\{2\nu^{-1} + \text{exp}(-\nu(1-y))(y - 1 - 2\nu^{-1})\right\} $$ Summing these results gives: $$\ \int_0^1 k(x,y) \;dxdy = \lambda^2 \left\{ 4\nu^{-1} + \text{exp}(-\nu(1-y))(y - 1 - 2\nu^{-1}) - \text{exp}(-\nu y)(y + 2\nu^{-1}) \right\}$$ Subbing in the value of $\ \nu$: $$\ \int_0^1 k(x,y) \;dxdy = \lambda^2 \left\{ \frac{4\ell}{\sqrt{3}} + \text{exp}\left(-\frac{\sqrt{3}}{\ell}(1-y)\right)\left(y - 1 - \frac{2\ell}{\sqrt{3}}\right) - \text{exp}\left(-\frac{\sqrt{3}}{\ell}y\right)\left(y + \frac{2\ell}{\sqrt{3}}\right) \right\}. $$

We can now integrate out $\ y$: $$\ \int_0^1 \int_0^1 k(x,y) \;dxdy = \int_0^1 \lambda^2 \left\{ \frac{4\ell}{\sqrt{3}} + \text{exp}\left(-\frac{\sqrt{3}}{\ell}(1-y)\right)\left(y - 1 - \frac{2\ell}{\sqrt{3}}\right) - \text{exp}\left(-\frac{\sqrt{3}}{\ell}y\right)\left(y + \frac{2\ell}{\sqrt{3}}\right) \right\}\;dy $$ $$\ =\lambda^2\left[ \frac{4\ell}{\sqrt{3}}y+ \frac{\ell}{\sqrt{3}}\text{exp}\left(-\frac{\sqrt{3}}{\ell}(1-y)\right)\left( y - 1 - \frac{3\ell}{\sqrt{3}}\right)+ \frac{\ell}{\sqrt{3}}\text{exp}\left(\frac{-\sqrt{3}}{\ell}y\right)\left(y + \frac{3\ell}{\sqrt{3}}\right) \right]_0^1 $$

$$\ = \frac{\lambda^2 \ell}{\sqrt{3}}\left[ 4y+ \text{exp}\left(-\frac{\sqrt{3}}{\ell}(1-y)\right)\left( y - 1 - \frac{3\ell}{\sqrt{3}}\right)+ \text{exp}\left(\frac{-\sqrt{3}}{\ell}y\right)\left(y + \frac{3\ell}{\sqrt{3}}\right) \right]_0^1 $$$$\ = \frac{\lambda^2\ell}{\sqrt{3}}\left\{ 4 - \frac{3\ell}{\sqrt{3}} + \text{exp}\left(-\frac{\sqrt{3}}{\ell}\right)\left(1 + \frac{3\ell}{\sqrt{3}}\right) - \frac{3\ell}{\sqrt{3}} - \text{exp}\left(-\frac{\sqrt{3}}{\ell}\right)\left(-1 -\frac{3\ell}{\sqrt{3}}\right) \right\} $$$$\ = \frac{\lambda^2\ell}{\sqrt{3}}\left\{ 4 - \frac{6\ell}{\sqrt{3}} + \text{exp}\left(-\frac{\sqrt{3}}{\ell}\right)\left(2 + \frac{6\ell}{\sqrt{3}}\right) \right\} $$

Now, integrating the mean: $$\ \int_0^1m(x)\;dx = \int_0^1 c \;dx = \left[cx\right]_0^1 = c $$

kernel_prior <- function(x, y, lambda_sq = 1, L = 1) {
    out1 <- lamba_sq * (1 + sqrt(3) * abs(x - y) / L)
    out2 <- exp(-sqrt(3) * abs(x - y) / L)
    return(lambda_sq * out1 * out2)
}
kernel_posterior <- function(lambda_sq =1, L = 1) {
    out = exp(-sqrt(3)/L)*(2+6*L/sqrt(3)) - 6*L/sqrt(3) + 4
    return(lambda_sq*L*out/sqrt(3))
}
